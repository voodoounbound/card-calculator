# HPMA Card Upgrade Calculator

A fan-made Harry Potter: Magic Awakened card upgrade calculator for estimating card copies, gold cost, wild cards, trade tokens, and leveling crystal savings.

## Features

- Card rarity auto-fill
- Upgrade cost calculator
- Gold cost tracking
- Normal and Refined Leveling Crystal allocation
- Calculation summary history
- CSV export and import
- Card image support
- GitHub Pages deployment
